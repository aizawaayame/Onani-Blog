## 1. Input

### 1.1. Rotation Input

`APawn` 下有三个添加旋转输入的接口 `APawn::AddControllerPitchInput`，`APawn::AddControllerYawInput`，`APawn::AddControllerRollInput`。

<Image src={_1Image} alt="Rotation Input调用链" />
<Steps>
1. `APawn::ForceRotation`：
    根据是否配置`bUserControllerRotationPitch(Roll, Yaw)`强制更新**ControlledPawn**的旋转，
2. `APlayerCameraManager::PrecessViewRotation`：
    更新**Camera**的旋转
</Steps>
