### 1.1. 题目简述

给定数组 `arr` 和一个值 `value`。原地移除等于 `value` 的元素，并返回移除后 `arr` 的 length。  
不能使用额外空间，时间复杂度要求为 $O(1)$

## 2. 解题思路

双指针：通过快慢指针在一个 for 循环下完成两个 for 循环的工作

## 3. Code

```cpp
int RemoveElement(int[] num, int val)
{
	int sidx = 0;
	for (int fidx = 0; fidx < num.Length; fidx++)
	{
		if (val != nums[fidx])
			nums[sidx++] = nums[fidx];
	}
	return sidx;
}
```
